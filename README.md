# 🤰 Pregnancy ML Models

임신 성공 여부를 예측하는 머신러닝 프로젝트입니다.  
IVF(체외수정) 및 DI(인공수정) 시술 데이터를 기반으로 임신 성공 여부(이진 분류)를 예측합니다.

---

### 수치형 변수 분포

- **총 생성 배아 수 / 저장된 배아 수**: 성공군의 중앙값이 실패군보다 높음 → 배아 수가 많을수록 성공에 유리
- **배아 이식 경과일**: 성공군에서 5일차(배반포기) 비율이 높음 → 5일 배반포 이식이 더 유리
- **이식된 배아 수**: 1~2개가 주류, 성공/실패군 간 분포 유사
- **난자 혼합 경과일**: 성공/실패 간 유의미한 차이 없음

## 🧠 모델링 전략

| 항목 | 권장 사항 |
|------|----------|
| 클래스 불균형 | SMOTE, class_weight='balanced', 또는 임계값 조정 |
| 고결측 변수 | 결측률 80% 이상 변수 제거 권장 |

| 다중공선성 | 난자/정자 군집 변수 PCA 또는 대표 변수 선택 |
| 핵심 피처 | 이식된 배아 수, 총 생성 배아 수, 연령대, 시술 유형 |
